# EliteHomes

A full-stack real estate web application developed using Spring Boot and MySQL that streamlines property buying, selling, and management through a secure and user-friendly platform. The application follows the MVC (Model–View–Controller) architecture to ensure scalability, maintainability, and efficient data management.

## Overview

EliteHomes is designed to replicate the functionality of modern real estate platforms by providing a centralized system where buyers can explore properties and sellers can manage listings. The application offers property search, detailed property information, wishlist management, agent communication, and seller analytics, creating a complete real estate ecosystem.

## Architecture

The project follows the MVC (Model–View–Controller) architecture:

* **Model:** Handles application data and database operations.
* **View:** User interface developed using HTML, CSS, and JavaScript.
* **Controller:** Processes user requests and manages application workflows.

This architecture ensures better code organization, maintainability, and scalability.

## Key Functionalities

### Property Search

Users can search for properties based on their preferred city and browse available listings efficiently.

### Property Listings

Sellers can add new properties, which are dynamically stored in the database and displayed on listing pages.

### Wishlist Management

Buyers can save properties to their wishlist for future consideration and comparison.

### Seller Analytics

Sellers can monitor property performance through view counts and wishlist statistics.

### Communication System

Buyers can directly connect with agents regarding specific properties, improving engagement and inquiry handling.
